# Password_Generator

A simple tool written in C that generates a secure password based on a user-specified length.

---

## Features

- Generate random passwords of any desired length.
- Includes uppercase, lowercase, numbers, and special characters for strong security.
- Lightweight and easy to compile.
- Open-source under the MIT License.

---

## Requirements

- A C compiler (e.g., `gcc`, `clang`).
- Compatible with Windows, Linux, and macOS.

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/Password_Generator.git
cd Password_Generator

2. Compile the program:

gcc passgenerator.c -o passgenerator

Usage

Run the program and follow the prompt to enter the desired password length:

./passgenerator


Example:

Enter the length of the password: 12
Generated password: aB3$d9K!pLq2
